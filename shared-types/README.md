# InCloser Shared Types

Shared TypeScript contracts used by `admin-web` and `admin-backend`.

## Build

- `npm install`
- `npm run build`

Exports include:

- `AdminUser`
- `DashboardSummary`
- `UserSummary`
- `FemaleModelSummary`
- `FemaleModelDetail`
- `AudioVerificationItem`
- `WithdrawalItem`
- `TransactionItem`
- `CmsBanner`
- `FaqItem`
- `AppSettings`
- `AuditLogItem`
- `ApiResponse<T>`
